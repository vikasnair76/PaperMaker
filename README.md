#PaperMaker

A new Flutter project.
PaperMaker is a lightweight web-based application that allows users to generate and customize wallpaper-style background designs directly in the browser. The project demonstrates dynamic DOM manipulation, real-time UI updates, and interactive frontend design using pure web technologies.

**Features**

Dynamic wallpaper pattern generation

Real-time design preview

Custom color selection

Adjustable layout controls

Fully browser-based (no backend required)

Lightweight and fast
